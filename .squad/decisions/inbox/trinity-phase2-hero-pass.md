# Decision: Phase 2 Hero Pass Grouped Delivery

- **Author:** Trinity
- **Date:** 2026-06-05T13:01:28.925-04:00
- **Status:** Executed

Issues #5, #6, and #7 are being delivered together on `squad/5-phase2-hero-pass` with one draft PR.

## Why

All three issues touch the same four hero surfaces (`Index`, `Manifesto`, `AI`, `ArchAssessment`) and need to be judged together as one visual system. Splitting them would create repeated edits in the same markup, increase merge risk, and make responsive verification harder because the acceptance criteria overlap.

## Scope

- Remove generic hero filler copy
- Standardize all four heroes on the Assessment-style structure
- Remove the floating hero diamond treatment that was colliding with the homepage video
- Move the homepage video out of the hero so the hero stays readable in one viewport
